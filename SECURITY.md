# Security Policy

Assumption Forge is currently a documentation, prompt, and skill repository. It does not ship a runtime service, package, or executable CLI yet.

Still, security matters because testing guidance can influence real projects.

## Reporting a concern

If you find guidance in this repository that could encourage unsafe behavior, weaken access-control testing, leak sensitive data, or create misleading confidence, please open an issue with:

- the affected file,
- the problematic guidance,
- the safer alternative you recommend,
- any relevant context.

If the concern is sensitive, avoid posting secrets, private code, credentials, or exploit details in public issues.

## Responsible examples

Examples in this repository should:

- use toy data,
- avoid real credentials,
- avoid real production endpoints,
- avoid private customer information,
- describe security-relevant behavior at a high level,
- focus on defensive testing quality.

## AI-generated contributions

AI-assisted contributions are welcome, but contributors are responsible for reviewing them.

Do not submit prompts, examples, or tests that create fake confidence or silently invent product requirements.
