# Code of Conduct

Assumption Forge should be a useful and welcoming project for people who care about better tests and better AI-assisted development.

## Expected behavior

Participants are expected to:

- be respectful,
- give constructive feedback,
- assume good intent when reasonable,
- focus criticism on ideas and contributions,
- welcome beginners,
- avoid harassment, insults, or personal attacks.

## Unacceptable behavior

Unacceptable behavior includes:

- harassment or discrimination,
- threats or intimidation,
- sexualized language or imagery,
- personal attacks,
- repeated disruption,
- publishing private information without permission.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or block participants who violate this code of conduct.

If you see a problem, open an issue or contact a maintainer through GitHub.

## Scope

This code applies to project spaces including issues, pull requests, discussions, and related community channels.
